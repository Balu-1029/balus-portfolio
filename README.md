# Balaji Kota — Personal Portfolio Website

A modern, responsive personal portfolio website built to showcase my academic background, technical skills, projects, internships, and contact information.

## Live Demo

Visit the website here: https://balajik-dev.vercel.app/

## Overview

This portfolio website serves as a digital resume and professional introduction. It is designed to present my profile in a clean, polished, and user-friendly way while highlighting my work in software development, data analysis, and web development.

## Features

- Responsive and modern UI
- Clean navigation across Home, About, Projects, and Contact pages
- Hero section with profile introduction and call-to-action buttons
- About section with academic background and technical skills
- Projects section showcasing completed work and internships
- Contact section with email, phone, LinkedIn, and GitHub links
- Smooth styling and hover effects using CSS

## Project Structure

```text
MY PORTFOLIO WEBSITE/
├── index.html          # Home page
├── about.html          # About me, skills, education
├── projects.html       # Projects and internship experience
├── contact.html        # Contact details
├── style.css           # Global styling for the website
└── Images/             # Profile image and icons used in the site
```

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive Web Design

## Pages Included

- Home: Introduction and quick access to projects and resume
- About: Background, skills, and education
- Projects: Featured projects and professional experience
- Contact: Direct ways to connect with me

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/Balu-1029/portfolio-website.git
   ```

2. Open the project folder.

3. Launch the website by opening `index.html` in your browser, or use a live server extension such as VS Code Live Server.
